# Ciclo de Vida del Agent

## Instalación

- Verifica permisos admin.
- Copia binarios.
- Crea servicio.
- Crea carpeta ProgramData.
- Configura puerto.
- Registra token.
- Abre firewall si se autoriza.

## Ejecución

- Inicia con Windows.
- Espera comandos.
- Envía heartbeat.
- Recolecta inventario programado.

## Actualización

- Recibe paquete.
- Verifica hash.
- Detiene servicio.
- Reemplaza binarios.
- Reinicia.
- Reporta versión.

## Desinstalación

- Detiene servicio.
- Quita servicio.
- Mantiene logs salvo confirmación.
