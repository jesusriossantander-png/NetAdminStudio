# Auditoría

## Auditoría

Toda acción administrativa debe responder:

- Quién lo hizo.
- Desde dónde.
- Sobre qué equipo.
- Qué cambió.
- Estado anterior.
- Estado nuevo.
- Resultado.
- Error si hubo.

## Acciones auditables

- Cambios de IP/DNS.
- Cambios de nombre.
- Cambios de permisos.
- Creación/baja de usuarios.
- Acciones masivas.
- Instalación/desinstalación de Agent.
