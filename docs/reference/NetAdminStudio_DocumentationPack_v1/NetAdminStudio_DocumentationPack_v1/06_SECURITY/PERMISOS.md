# Permisos del Sistema

## Roles

- SuperAdmin
- Admin
- Technician
- Auditor
- Viewer

## Matriz resumida

SuperAdmin: todo.  
Admin: administración general.  
Technician: red, impresoras, servicios.  
Auditor: lectura e informes.  
Viewer: solo lectura limitada.

## Regla

Cada comando remoto debe declarar permiso requerido.
