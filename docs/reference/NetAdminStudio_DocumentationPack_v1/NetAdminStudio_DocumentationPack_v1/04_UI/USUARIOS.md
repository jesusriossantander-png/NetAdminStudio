# Módulo Usuarios

## Estilo UI

- Menú lateral.
- Encabezado claro por módulo.
- Tarjetas de estado.
- Tabla central.
- Panel derecho de detalle.
- Botones principales arriba.
- Estados: verde correcto, amarillo advertencia, rojo error, azul acción.

## Regla UX

Primero diagnóstico, luego acción.

## Vistas

- Usuarios locales.
- Grupos locales.
- Accesos por usuario.
- Alta/Baja empleado.

## Acciones

- Crear usuario.
- Deshabilitar.
- Activar.
- Cambiar contraseña.
- Agregar/quitar de grupo.
- Aplicar perfil.
