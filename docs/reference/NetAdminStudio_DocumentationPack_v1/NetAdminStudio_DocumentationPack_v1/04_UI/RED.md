# Módulo Red

## Estilo UI

- Menú lateral.
- Encabezado claro por módulo.
- Tarjetas de estado.
- Tabla central.
- Panel derecho de detalle.
- Botones principales arriba.
- Estados: verde correcto, amarillo advertencia, rojo error, azul acción.

## Regla UX

Primero diagnóstico, luego acción.

## Pantalla Red

Columnas:
- Estado
- IP
- Nombre
- MAC
- Tipo
- Latencia
- Puertos
- Agente
- Último contacto

Acciones:
- Descubrir red.
- Ping.
- Probar puerto.
- Resolver nombre.
- Instalar Agent.
- Abrir panel de PC.
