# Operations Center

## Estilo UI

- Menú lateral.
- Encabezado claro por módulo.
- Tarjetas de estado.
- Tabla central.
- Panel derecho de detalle.
- Botones principales arriba.
- Estados: verde correcto, amarillo advertencia, rojo error, azul acción.

## Regla UX

Primero diagnóstico, luego acción.

## Objetivo

Ser la torre de control.

## Componentes

- Mapa lógico de red.
- Lista de incidentes.
- Health Score general.
- Acciones recomendadas.
- Actividad reciente.
- Top riesgos.

## Mensaje inicial

“Detecté X problemas que conviene revisar.”
