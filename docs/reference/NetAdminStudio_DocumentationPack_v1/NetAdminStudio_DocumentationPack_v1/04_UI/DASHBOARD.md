# Dashboard

## Estilo UI

- Menú lateral.
- Encabezado claro por módulo.
- Tarjetas de estado.
- Tabla central.
- Panel derecho de detalle.
- Botones principales arriba.
- Estados: verde correcto, amarillo advertencia, rojo error, azul acción.

## Regla UX

Primero diagnóstico, luego acción.

El Dashboard será una vista resumida. En versiones posteriores el nombre principal será Operations Center.

Mostrar:
- PCs online/offline.
- Agentes instalados.
- Alertas.
- Incidentes abiertos.
- Impresoras con problemas.
- Carpetas inseguras.
- Último escaneo.
