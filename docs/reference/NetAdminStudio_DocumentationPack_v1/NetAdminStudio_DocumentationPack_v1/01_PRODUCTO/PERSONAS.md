# Personas y Roles

## Administrador Principal
Puede realizar todas las acciones. Usa la consola completa.

## Técnico
Puede diagnosticar, administrar impresoras, red y servicios. No puede modificar permisos críticos sin autorización.

## Supervisor
Solo lectura, informes y auditorías.

## Usuario final
No usa la consola. Interactúa indirectamente con Agent.

## Roles internos del sistema

- SuperAdmin
- Admin
- Technician
- Auditor
- Viewer

Cada acción debe validar rol.
