# Roadmap

## Roadmap

### Fase 0 - Documentación
- Estructura de documentación.
- Arquitectura base.
- Prompts para Claude Code.
- Decisiones ADR.

### Fase 1 - MVP técnico
- Solución .NET 8.
- Console WPF.
- Agent Windows Service.
- Core.
- Data SQLite.
- Descubrimiento básico.
- Registro de assets.
- Comunicación básica Console-Agent.

- Spooler.
- Compartir impresoras.
- Instalar impresoras compartidas.
- IP fija/DHCP/DNS.
- Diagnóstico de red.

- Permisos NTFS.
- Permisos por archivo/subcarpeta.
- Usuarios y grupos locales.
- Alta/baja de empleados.
- Plantillas.

### Fase 4 - Auditoría e informes
- Health Score.
- Incidentes.
- Seguridad de recursos compartidos.
- Informes PDF/HTML/TXT.
- Acciones masivas.

### Fase 5 - Producto avanzado
- Instalador profesional.
- Updater.
- Plugins.
- Panel web.
- IA asistente.
- App móvil.
