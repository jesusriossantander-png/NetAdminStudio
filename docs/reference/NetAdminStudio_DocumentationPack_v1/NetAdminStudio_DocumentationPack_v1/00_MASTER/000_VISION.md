# Visión del Proyecto

**Producto:** NetAdmin Studio Enterprise  
**Versión documentación:** 1.0  
**Objetivo:** servir como documentación completa inicial para que Claude Code trabaje dentro de VS Code con contexto claro, arquitectura modular y reglas de calidad.

## Reglas globales del proyecto

- Diagnosticar antes de modificar.
- Confirmar cambios críticos.
- Crear backup antes de modificar permisos, recursos compartidos o configuración de red.
- Registrar toda acción en logs y auditoría.
- Separar UI, lógica de negocio, acceso a datos y comunicación.
- No depender de Internet para funciones críticas.
- Priorizar seguridad sobre comodidad.
- Mantener compatibilidad con Windows 10 y Windows 11.
- Diseñar pensando en crecimiento futuro: Linux, Raspberry, NAS, Docker, Hyper-V, Active Directory y plugins.


## Visión

NetAdmin Studio Enterprise será una plataforma profesional para administrar infraestructura Windows desde una única consola central, con posibilidad de crecer hacia otros sistemas y dispositivos.

La visión no es crear un simple configurador de impresoras. El producto debe convertirse en una torre de control para oficinas pequeñas y medianas.

## Frase guía

> Todo el control de tu infraestructura, desde un solo lugar.

## Problemas que resuelve

- PCs con configuraciones de red desordenadas.
- Impresoras USB compartidas difíciles de administrar.
- Carpetas compartidas con permisos inseguros.
- Usuarios locales creados sin criterio.
- Falta de inventario actualizado.
- Ausencia de historial de cambios.
- Diagnóstico lento de problemas comunes.
- Necesidad de moverse físicamente de PC en PC.

## Resultado esperado

El administrador debe poder abrir la consola y saber:

- Qué equipos están online.
- Qué equipos tienen problemas.
- Qué impresoras están compartidas.
- Qué carpetas están expuestas.
- Qué usuarios tienen acceso a cada recurso.
- Qué cambió recientemente.
- Qué incidentes requieren atención.
