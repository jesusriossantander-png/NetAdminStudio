# ADR 0001 - Console + Agent

## Decisión

Usar arquitectura Console + Agent en lugar de PowerShell remoto puro.

## Motivo

Reduce problemas de permisos, permite monitoreo, inventario persistente y acciones locales confiables.

## Consecuencias

- Documentar la decisión.
- Implementar interfaces extensibles.
- No acoplar módulos a implementaciones temporales.
