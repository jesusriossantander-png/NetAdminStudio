# Prompt — Revisar Pack 4B

Revisar como arquitecto de auditoría, reporting y seguridad.

Buscar:

- eventos mutables;
- falta de correlación;
- hashes no verificados;
- datos sensibles;
- reportes no reproducibles;
- falta de versionado;
- SQL libre;
- exportaciones sin permisos;
- timezone incorrecto;
- pérdida de evidencia;
- retención peligrosa;
- purgas no auditadas;
- fallos de paginación;
- consultas lentas;
- N+1;
- archivos temporales;
- reportes duplicados;
- scheduler no idempotente;
- falta de pruebas.

Entregar severidad, evidencia, impacto y corrección.
