# Retención y archivo

## Clases de datos

- auditoría crítica;
- auditoría operativa;
- métricas;
- trabajos;
- reportes;
- evidencias;
- exportaciones;
- logs técnicos.

## Política

- retención configurable;
- mínimo por categoría;
- legal hold futuro;
- archivo;
- compactación;
- borrado seguro solo donde corresponda;
- auditoría de purga.

## Regla crítica

Los eventos de auditoría críticos no se eliminan mediante limpieza rutinaria.
