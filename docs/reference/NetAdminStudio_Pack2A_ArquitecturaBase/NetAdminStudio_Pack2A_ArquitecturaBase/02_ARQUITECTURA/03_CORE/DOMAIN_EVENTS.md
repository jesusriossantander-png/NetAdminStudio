# Eventos de Dominio

## Eventos iniciales

- AssetDiscovered
- AssetUpdated
- AgentRegistered
- PrinterShared
- PrinterUnshared
- ShareCreated
- PermissionChanged
- UserCreated
- UserDisabled
- IncidentOpened
- IncidentResolved

## Propiedades comunes

- EventId
- AssetId
- OccurredAt
- Actor
- CorrelationId
- Severity
- Payload

## Uso

Los eventos sirven para:
- Historial.
- Auditoría.
- Alertas.
- Automatizaciones futuras.
