# Notas de persistencia

- SID es la clave de identidad real.
- Los snapshots de permisos son inmutables.
- ACL crudas se conservan para trazabilidad.
- Los nombres de archivos abiertos deben protegerse.
- Las membresías anidadas pueden almacenarse como cache calculada.
- Los planes de cambio deben guardar solicitud, vista previa, respaldo y resultado.
- Nunca guardar credenciales de dominio en estas tablas.
