# Inventario Completo e Incremental

## Full Snapshot

Se envía:
- al enrolar,
- manualmente,
- periódicamente,
- después de actualización importante.

## Delta

Solo cambios desde último snapshot:
- IP.
- impresoras.
- software.
- usuarios.
- servicios.

## Regla

La Console puede solicitar Full si pierde continuidad.
