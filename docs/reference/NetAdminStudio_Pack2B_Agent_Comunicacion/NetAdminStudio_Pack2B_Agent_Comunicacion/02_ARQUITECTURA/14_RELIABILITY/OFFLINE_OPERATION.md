# Operación Offline

## Agent sin Console

Debe:
- seguir ejecutando Jobs ya aceptados,
- guardar resultados,
- recolectar inventario según política,
- mantener logs,
- reenviar cuando la Console vuelva.

## Console sin Agent

Debe:
- mostrar último estado conocido,
- marcar datos como desactualizados,
- no simular éxito.
