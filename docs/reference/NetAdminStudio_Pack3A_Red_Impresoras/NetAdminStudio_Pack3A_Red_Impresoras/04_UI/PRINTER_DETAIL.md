# UI — Detalle de impresora

## Tabs

- Resumen
- Estado
- Colas
- Trabajos
- Consumibles
- Contadores
- Mantenimiento
- Historial
- Documentos
- Auditoría

## Resumen

- nombre;
- modelo;
- serie;
- conexión;
- host;
- IP/URI;
- driver;
- ubicación;
- capacidades;
- estado;
- última actualización.

## Acciones

- actualizar;
- abrir administración web;
- pausar/reanudar cola;
- registrar mantenimiento;
- reemplazar consumible;
- retirar.

Las operaciones sobre colas deben mostrar alcance, impacto y resultado.
