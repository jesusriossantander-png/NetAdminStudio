# UI — Dashboard de red

## Encabezado

- título;
- alcance actual;
- última actualización;
- botón actualizar;
- botón nuevo escaneo;
- estado del agente.

## KPI

- activos totales;
- online;
- degradados;
- offline;
- alertas abiertas;
- latencia promedio;
- cambios recientes.

## Secciones

1. Estado general.
2. Alertas prioritarias.
3. Disponibilidad por sector.
4. Tráfico observado.
5. Cambios de topología.
6. Dispositivos recientemente descubiertos.
7. Jobs activos.

## Interacción

- cada KPI filtra;
- las gráficas muestran rango temporal;
- la fecha de actualización siempre visible;
- skeleton durante carga;
- estado vacío explicativo;
- error recuperable con reintento.
