# Checklist de aceptación

- [ ] Las identidades usan SID.
- [ ] Local y dominio están separados.
- [ ] NTFS y SMB no se mezclan.
- [ ] Se calcula acceso efectivo.
- [ ] La UI explica el origen del acceso.
- [ ] Se diferencian reglas heredadas y explícitas.
- [ ] Toda modificación posee vista previa.
- [ ] Existe respaldo y rollback.
- [ ] Las operaciones masivas son jobs.
- [ ] No se ejecutan comandos libres.
- [ ] Las credenciales no se almacenan en texto plano.
- [ ] Los nombres de archivos abiertos están protegidos.
- [ ] Existe auditoría completa.
- [ ] Las operaciones sensibles requieren RBAC.
