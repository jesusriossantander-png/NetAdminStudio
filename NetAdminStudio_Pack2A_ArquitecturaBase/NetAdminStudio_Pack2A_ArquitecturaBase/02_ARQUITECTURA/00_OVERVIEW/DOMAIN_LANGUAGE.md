# Lenguaje de Dominio

## Términos oficiales

### Asset
Activo administrado: PC, impresora, router, NAS, etc.

### Managed Node
Asset con Agent instalado y administrable.

### Observation
Dato recolectado sin modificar el sistema.

### Command
Solicitud explícita para realizar una acción.

### Event
Hecho registrado en el sistema.

### Incident
Problema detectado que requiere seguimiento.

### Policy
Estado deseado aplicable a uno o más assets.

### Health Score
Indicador calculado de salud.

### Evidence
Dato que respalda una alerta o incidente.

## Regla de nomenclatura

El código y la documentación deben utilizar estos términos de forma consistente.
