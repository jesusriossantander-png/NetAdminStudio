# Sistema de Plugins

## Objetivo futuro

Permitir agregar módulos sin reescribir núcleo.

## Plugins futuros

- Active Directory
- Linux Agent
- Raspberry
- Docker
- Hyper-V
- VMware
- NAS
- Backup
- IA

## Contrato

Cada plugin debe declarar:
- Nombre
- Versión
- Permisos
- Vistas
- Comandos
- Modelos soportados
