# Arquitectura de Seguridad

## Principios

- Confianza mínima.
- Agente solo acepta consola autorizada.
- Logs obligatorios.
- Confirmación para acciones destructivas.
- Backups previos.

## MVP

- Token de administración generado en instalación.
- Lista blanca opcional de IPs de consola.
- Roles locales en Console.

## Futuro

- Certificados por instalación.
- mTLS.
- Rotación de tokens.
- Integración AD.
