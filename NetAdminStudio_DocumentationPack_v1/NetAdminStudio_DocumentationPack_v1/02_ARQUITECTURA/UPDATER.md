# Actualizador

## Objetivo

Actualizar Console y Agents sin intervención manual extensa.

## Fases

### MVP
Actualización manual con instalador.

### Fase 2
Consola distribuye nuevo Agent.

### Fase 3
Updater con rollback.

## Reglas

- Nunca actualizar sin verificar hash.
- Mantener versión anterior.
- Registrar resultado.
