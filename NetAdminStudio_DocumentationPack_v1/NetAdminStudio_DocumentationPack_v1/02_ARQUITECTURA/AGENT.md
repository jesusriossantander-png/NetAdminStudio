# NetAdmin Agent

## Responsabilidad

Servicio Windows instalado en cada PC administrada.

## Funciones

- Reportar inventario.
- Ejecutar comandos autorizados.
- Administrar impresoras.
- Administrar carpetas compartidas.
- Administrar usuarios.
- Reportar eventos.
- Recibir actualizaciones.

## Características

- Inicia con Windows.
- Corre en segundo plano.
- Consume pocos recursos.
- Registra logs locales.
- Solo acepta comandos autenticados.

## Servicio

Nombre sugerido:
`NetAdminAgent`

Puerto sugerido:
`47820` configurable.
