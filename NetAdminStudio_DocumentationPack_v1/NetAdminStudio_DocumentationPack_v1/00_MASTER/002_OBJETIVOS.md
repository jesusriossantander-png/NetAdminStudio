# Objetivos

## Objetivo principal

Crear una plataforma de administración remota para infraestructura Windows basada en Console + Agent.

## Objetivos funcionales

- Descubrir equipos en red.
- Instalar/verificar agentes.
- Inventariar hardware y software.
- Administrar impresoras locales y compartidas.
- Administrar carpetas compartidas.
- Administrar permisos SMB y NTFS.
- Administrar usuarios y grupos locales.
- Aplicar políticas por sector o perfil.
- Ejecutar diagnósticos.
- Crear reportes.
- Mantener historial.
- Detectar incidentes.
- Permitir acciones masivas.

## Objetivos técnicos

- Arquitectura modular.
- WPF con MVVM.
- .NET 8.
- Windows Service para Agent.
- SQLite local.
- Comunicación segura Console-Agent.
- Logs estructurados.
- Posibilidad futura de plugins.
