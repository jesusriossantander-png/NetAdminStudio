# Permisos NTFS

## Principio

NTFS define acceso real a archivos y subcarpetas.

## Acciones

- Leer ACL.
- Crear backup.
- Modificar ACL.
- Restaurar ACL.
- Ver herencia.
- Quitar herencia con advertencia.
- Ver permisos efectivos.

## Advertencias

No recomendar permisos por archivo salvo casos puntuales. Preferir subcarpetas.
