# Requisitos Funcionales y No Funcionales

## Requisitos funcionales

- Descubrimiento LAN.
- Comunicación con Agent.
- Inventario.
- Administración remota.
- Logs.
- Auditoría.
- Backups.
- Reportes.
- Gestión de permisos.

## Requisitos no funcionales

- Rendimiento: Agent debe usar mínimo CPU/RAM.
- Seguridad: autenticación obligatoria.
- Confiabilidad: no romper permisos existentes.
- Mantenibilidad: arquitectura modular.
- Usabilidad: interfaz clara y guiada.
- Compatibilidad: Windows 10/11.
- Operación offline: todo en LAN.
