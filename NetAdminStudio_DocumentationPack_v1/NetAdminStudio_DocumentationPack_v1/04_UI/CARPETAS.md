# Módulo Carpetas Compartidas

## Estilo UI

- Menú lateral.
- Encabezado claro por módulo.
- Tarjetas de estado.
- Tabla central.
- Panel derecho de detalle.
- Botones principales arriba.
- Estados: verde correcto, amarillo advertencia, rojo error, azul acción.

## Regla UX

Primero diagnóstico, luego acción.

## Vista explorador

Mostrar árbol:
- Recurso compartido.
- Subcarpetas.
- Archivos.

Panel derecho:
- Ruta.
- Propietario.
- Herencia.
- Permisos SMB.
- Permisos NTFS.
- Usuarios que pueden ver.

Acciones:
- Crear recurso.
- Quitar recurso.
- Cambiar permisos.
- Backup ACL.
- Restaurar ACL.
