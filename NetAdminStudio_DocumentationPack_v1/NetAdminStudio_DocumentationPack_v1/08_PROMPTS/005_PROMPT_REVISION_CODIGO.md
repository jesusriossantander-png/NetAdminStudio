# Prompt Revisión de Código

Revisá el código generado.

Verificá:
- Separación de responsabilidades.
- MVVM correcto.
- Manejo de errores.
- Logs.
- Seguridad.
- No hay secretos hardcodeados.
- No se ejecutan acciones peligrosas sin confirmación.
- La solución compila.

Generá lista de correcciones antes de modificar.
