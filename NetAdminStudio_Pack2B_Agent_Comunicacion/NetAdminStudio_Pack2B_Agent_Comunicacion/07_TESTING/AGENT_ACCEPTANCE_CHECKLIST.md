# Checklist de Aceptación del Agent

- Se instala como Windows Service.
- Inicia con Windows.
- No muestra UI.
- Responde `/health`.
- Protege endpoints.
- Envía heartbeat.
- Devuelve capabilities.
- Recolecta inventario.
- Ejecuta comando de prueba.
- Persiste Job.
- Recupera Job después de reinicio.
- Registra logs.
- No guarda secretos en texto plano.
