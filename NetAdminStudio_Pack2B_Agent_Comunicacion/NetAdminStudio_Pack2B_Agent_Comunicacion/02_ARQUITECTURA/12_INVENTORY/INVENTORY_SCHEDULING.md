# Programación de Inventario

## Frecuencias sugeridas

- Básico: cada 5 minutos.
- Hardware: cada 24 horas.
- Software: cada 12 horas.
- Impresoras: cada 2 minutos.
- Shares/usuarios: cada 15 minutos.
- Inventario completo: cada 24 horas.
