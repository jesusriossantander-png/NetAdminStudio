# Autenticación Console-Agent

## MVP

- Token por instalación.
- Token protegido con DPAPI.
- Encabezado firmado o Bearer token.
- Lista blanca opcional de IPs.

## Futuro

- Certificado por Agent.
- mTLS.
- Rotación automática.
- Console Authority.

## Regla

El endpoint de administración no puede operar anónimamente.
